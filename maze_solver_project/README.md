# Maze Solver Project

This project is designed to solve mazes using different algorithms such as BFS, DFS, and A*. It also includes a GUI interface for easy maze-solving.

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-repository/maze-solver.git
   cd maze-solver
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the project:**
   ```bash
   python main.py
   ```

## Configuration

Edit `config.yaml` to adjust maze settings, solver method, and delay between steps.

## Usage

1. Drop a maze image into the interface window.
2. Select the maze-solving algorithm (BFS, DFS, or A*).
3. Click "Solve Maze" to start solving.
